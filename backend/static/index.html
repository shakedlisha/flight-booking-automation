Booking form demo

Use the Flight booking paste helper extension popup on this page: paste text (or PDF there), then Process — fields below should fill if the API and extension options are set correctly.

Checklist

  1. API running at http://127.0.0.1:8000 — easiest: double-click Start-API.bat in the project folder (starts server + opens this page).
  2. In backend/.env: set GEMINI_API_KEY; add your extension ID to CORS_ORIGINS as chrome-extension://….
  3. Extension Options: API base URL http://127.0.0.1:8000; bearer token must match EXTRACT_BEARER_TOKENS in .env if you use it.
  4. Click the extension icon → paste sample text below (or your own) → Process.
